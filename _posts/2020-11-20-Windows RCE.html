An RCE vulnerability was found in Windows DNS Server.

The scenarios are as follows:

1 : Some users ask the target server DNS questions about the attacker's server under the influence of the attacker.


2 : Target server ask "who is master dns server of www.attacker.com"

3 : Through the above process, the target server queries the IP of www.attacker.com to the server of the attacker.

4 : Attacker transfers payload of RCE vulnerability

For a detailed description of the vulnerability, see the link below.
